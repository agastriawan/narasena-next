# Checklist SEO untuk Website Nara Sena Advertising

## 🎯 Fase Setup Awal (Minggu Pertama)

### Google Search Console
- [ ] Buka https://search.google.com/search-console
- [ ] Tambahkan property website: https://narasena.com
- [ ] Verifikasi kepemilikan (metode HTML tag sudah disiapkan di code)
- [ ] Submit sitemap: https://narasena.com/sitemap.xml
- [ ] Cek coverage report untuk memastikan semua halaman terindex

### Google Business Profile (GMB) - PRIORITAS TINGGI!
- [ ] Buat akun di https://business.google.com
- [ ] Isi informasi lengkap:
  - [ ] Nama: Nara Sena Advertising
  - [ ] Kategori: Sign Shop / Advertising Agency
  - [ ] Alamat lengkap & pin map yang tepat
  - [ ] No. telepon: 0812-8600-3110
  - [ ] Website: https://narasena.com
  - [ ] Jam buka: Senin-Sabtu 07:00-21:00
- [ ] Verifikasi bisnis (tunggu kartu pos dari Google)
- [ ] Upload minimal 10 foto berkualitas tinggi:
  - [ ] Logo perusahaan
  - [ ] Foto kantor/workshop
  - [ ] 5-7 foto hasil karya terbaik
  - [ ] Foto tim bekerja

### Google Analytics
- [ ] Buat akun Google Analytics 4
- [ ] Dapatkan Measurement ID (format: G-XXXXXXXXXX)
- [ ] Buat file `.env.local` dan tambahkan:
  ```
  NEXT_PUBLIC_GA_ID=G-XXXXXXXXXX
  ```
- [ ] Test tracking dengan Real-Time report

### Update Verification Code
- [ ] Buka Google Search Console
- [ ] Copy verification code
- [ ] Edit file `app/layout.tsx`
- [ ] Ganti `'your-google-verification-code'` dengan code asli
- [ ] Deploy website
- [ ] Klik "Verify" di Google Search Console

## 🚨 PENTING - Jangan Lakukan Ini!

❌ Keyword stuffing (mengulang keyword terlalu banyak)
❌ Hidden text/links
❌ Beli backlinks murah dari Fiverr
❌ Spin artikel dari website lain
❌ Fake reviews
❌ Cloaking atau redirect manipulatif
❌ Duplicate content
